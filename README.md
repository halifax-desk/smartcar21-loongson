## 项目结构

```
SmartCarVision21/
├── Python_Script/       # Python 巡线算法（主力开发环境）
├── bin_test/            # C++ 编译测试（从 Python 向龙芯平台移植）
├── Camera_recode/       # 摄像头录制素材（不同角度、倾斜测试）
├── Demo_Video/          # 算法演示录像
└── Documents/           # 参考资料（视觉算法、Voronoi 图、十字补线等）
```

## 各模块简介

### Python_Script — 算法原型

所有巡线算法的 Python 实现，上手快、调试方便。包含多种方案：

- **二值化扫描**：`centerline_by_binary.py`，基于阈值二值化，从白色像素最多列向两侧扫描边界
- **Canny 边缘检测**：`edge_detection.py` / `centerline_by_canny.py`，基于 Canny 算子提取边缘后计算中心线
- **2ROI 双区域扫描**：`2ROI_Scan.py`，在两个 ROI 区域内逐行查找左右边界，实测可用
- **八邻域巡线（8Area）**：参考上交大开源方案实现的核心巡线骨架，从底部向上逐行追踪赛道边界
- **补线与转折点**：`Fill_Line.py`、`Turing_point.py`，处理十字路口、断线等特殊情况

### bin_test — C++ 移植测试

将 Python 算法移植到 C++（目标平台：龙芯 + OpenCV），包含摄像头实时采集与巡线可视化。是 Python → 龙芯嵌入式平台的中间过渡阶段。

### Camera_recode / Demo_Video

用于离线测试与效果展示的录制素材，模拟不同工况（倾斜、竖屏等）下的摄像头输入。

### Documents

比赛中收集的视觉算法参考资料，包括圆环处理、十字补线、最小二乘法拟合、Voronoi 图等。

## 环境

- **Python 方案**：Python 3.x + OpenCV + NumPy
- **C++ 方案**：OpenCV 4.x + g++ / MSVC
